When editing a link file 'link_(branch name).json' the format is as followed.

```` json
[
    {
        "title": "Air Force Weather Agency",                                                    <--- The text that is displayed
        "url": "https://weather.af.mil/",                                                       <--- The url to the site
        "description": "Air Force Weather Agency is the weather presented by the Air Force.",   <--- The description for the link
        "cardId": 23                                                                            <--- the card id this can be found by looking at the name of the card file
    },
    {...}
]

````
